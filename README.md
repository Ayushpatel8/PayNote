# PayNote

Accounting & Billing System built using Laravel.

## About PayNote
PayNote is a web-based application designed to manage accounting, billing, and financial records efficiently.

## Tech Stack
- Laravel
- PHP
- MySQL
- HTML, CSS, JavaScript

## Installation
1. Clone the repository
2. Run `composer install`
3. Configure `.env`
4. Run migrations
5. Start the server
